Arch Instalação.
GUIA RAPIDO
INSTALAR ARCH!

1. Carregando teclado.

# loadkeys br-abnt2

2. Verificar o modo de inicialização com o comando.

# ls /sys/firmware/efi/efivars


Se o diretório não existir, o sistema
pode ser inicializado no modo BIOS.

3. Acertando data e hora.

# timedatectl
# timedatectl set-ntp true
# timedatectl set-timezone America/Belem
# timedatectl status

4. formatando disco.

Se for BIOS use # mkfs.ext2 /dev/sda1
Se for UEFI use # mkfs.fat -F32 -n BOOT /dev/sda1
# mkfs.ext4 /dev/sda3
# mkswap /dev/sda2
# swapon /dev/sda2

5. Verificando internet.

# ip -s -c -h a
# wifi-menu +(nome do dispositivo encontrado pelo comando acima)

6. Montando.

# mkdir /mnt/boot
# mount /dev/sda1 /mnt/boot
# mount /dev/sda3 /mnt

7. Acertando horário.

# rm /etc/localtime
# ln -s /usr/share/zoneinfo/America/Belem /etc/localtime
# hwclock --systohc
# date

8. Baixando base Arch Linux.

# pacstrap -i /mnt base base-devel linux linux-firmware

9. Gerando fstab.

# genfstab -U /mnt >> /mnt/etc/fstab
# cat /mnt/etc/fstab

9.1. Edite o fstab apagando o uuid.

# vi /mnt/etc/fstab

10. Entrando no chroot.

# arch-chroot /mnt

11. Alterando idioma.

# vi /etc/locale.gen
Descomente a linha pt_BR.UTF-8 UTF-8
# locale-gen
# echo LANG=pt_BR.UTF-8 > /etc/locale.conf
# export LANG=pt_BR.UTF-8

12. Nomeando a maquina.

# echo nomeEscolhido > /etc/hostname
# cat /etc/hostname
# vi /etc/hosts // [configure aqui também]

13. Instalando pacotes necessário para o
funionamento do wifi. (para notebooks)

# pacman -S wireless_tools iwd wpa_supplicant dialog

14. mk init cp io configurando ambiente.

# mkinitcpio -P

15. Adicionando senha do root.

# passwd

16. Adicionando multilib/yaourt.

# vi /etc/pacman.conf
descomente as linhas
[multilib]
inclue = /etc/pacman.d/mirrorlist
acrescente as linhas abaixo para a instalação do yaourt
[archlinuxfr]
SigLevel = Never
Server = http://repo.archlinux.fr/$arch

17. Configurando teclado terminal.

# vi /etc/vconsole.conf
KEYMAP=br-abnt2

18. Atualizando o sistema.

# pacman -Sy

19. Instalando o grub (modo BIOS) .

# pacman -S grub
# grub-install --target=i386-pc --boot-directory=/boot /dev/sda
# grub-mkconfig -o /boot/grub/grub.cfg


Caso o grub apresente erro faça esta segunda opção:

# mkdir /mnt/boot
# mount /dev/sda1 /mnt/boot
# pacman -S grub
# grub-install --target=i386-pc --boot-directory=/mnt/boot /dev/sda
# grub-mkconfig -o /boot/grub/grub.cfg

20. Instalando o grub (modo UEFI) .

# pacman -S grub-efi-x86_64 efibootmgr
# lsblk
# mkdir /boot/efi
# mount /dev/sda1 /boot/efi
# grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=BOOT --recheck
# grub-mkconfig -o /boot/grub/grub.cfg

21. Criando usuário.

# useradd -m -G users,wheel,audio,daemon,dbus,disk,games,network,rfkill,optical,power,storage,scanner,lp,video -s /bin/bash nomeUsuario

# passwd nomeUsuario

22. Instalando pacotes essenciais.

# pacman -S acpi acpid // (para notebook bateria)
alsa-plugins alsa-utils pulseaudio-alsa // [Caso o alsa não fucione]
gst-plugins-{base,good,bad,ugly} gst-libav
rxvt-unicode urxvt-perls dhcpcd
firefox firefox-i18n-pt-br
git wget
OBS 1: Se você não utiliza wifi use o comando dhcpcd + nome do dispisitivo (Ex: wlp3s0).

23. Dando permissão ao usuário

# vi /etc/sudoers
abaixo da linha root ALL=(ALL) ALL adicione
nomeUsuario ALL=(ALL) ALL

24. Saindo

# exit
# umount -R /mnt
# reboot

25. Iniciando a internet no boot.

# ip link

$ sudo systemctl start dhcpcd@eth0.service
$ sudo systemctl enable dhcpcd@eth0.service
#$ sudo systemctl start NetworkManager
#$ sudo systemctl enable NetworkManager


troque eth0 pelo seu dispositivo. # vi /etc/resolv.conf

         nameserver 8.8.8.8
         nameserver 8.8.4.4

26. Criando arquivo init.

echo 'exec [wm/de]' > /home/USUARIO/.xinitrc

27. Instalando Xorg.

# pacman -S xorg-xinit xorg-server ttf-dejavu
Radeon extra/xf86-video-amdgpu
Intel extra/xf86-video-intel
Nvidia extra/nvidia

Teclado extra/xf86-input-keyboard
Mouse extra/xf86-input-mouse
touchpads (Notebooks) extra/xf86-input-synaptics

28. Configurando teclado Xorg.

# vi /etc/X11/xorg.conf.d/00-keyboard.conf

Section "InputClass"
        Identifier "system-keyboard"
        MatchIsKeyboard "on"
         Option "XkbLayout" "br"
         Option "XkbVariant" ",abnt2"
         Option "XkbOptions" "grp:shift_toggle,grp_led:scroll"
EndSection

29. Removendo Tearing da placa de video.

Intel
# vi /usr/share/X11/xorg.conf.d/20-intel.conf

Section "Device"
        Identifier "Intel Graphics"
        Driver "intel"
         Option "AccelMethod" "sna"
         Option "TearFree" "on"
         Option "DRI" "3"
EndSection


Radeon
# vi /usr/share/X11/xorg.conf.d/50-device.conf

Section "Device"
        Identifier "Default"
        Driver "amdgpu"
         Option "DRI" "3"
         Option "TearFree" "True"
EndSection
